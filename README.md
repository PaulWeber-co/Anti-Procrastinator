# 📋 Habit Tracker — Notion Style

Eine ästhetische, minimalistische Habit-Tracker Web-App im Notion-Design.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## ✨ Features

- **Echtzeituhr** — Aktuelle Uhrzeit & Datum (Europe/Berlin)
- **Wetter-Widget** — Live-Wetter via wttr.in (kein API-Key nötig)
- **To-Do Liste** — Tasks erstellen, abhaken, löschen mit Kategorien & Datumszuweisung
- **Monatskalender** — Interaktiver Kalender mit farbigen Task-Indikatoren
- **Fortschritts-Diagramme** — Wöchentliche Abschlussrate (Liniendiagramm) & Kategorieverteilung (Donut-Chart)
- **Statistik-Karten** — Überblick über Gesamt, Erledigt, Offen und Abschlussrate
- **Dark/Light Mode** — Toggle mit gespeicherter Präferenz
- **Lokale Datenspeicherung** — Alle Daten werden im Browser (localStorage) gespeichert
- **Responsive Design** — Funktioniert auf Desktop & Mobile

## 🚀 Nutzung

Einfach `index.html` im Browser öffnen — fertig!

```bash
open index.html
```

Oder mit einem lokalen Server:

```bash
npx serve .
```
